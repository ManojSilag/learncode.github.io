Palindrome Program

A Palindrome number is a which remains the same on reversal. For example, some palindrome numbers are 121, 12321. To check whether a number is a palindrome or not first we reverse it and then compare the number obtained with the original number, if both are same then the number is palindrome otherwise not. This are a simple Program that accepts an Input number from User and Checks whether the number is Palindrome or not.
